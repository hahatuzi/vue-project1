# vue-project3
### 前后端交互设计，axios的设计包登陆判断，交互请求处理和相应处理
### 设计路由，路由的设计包括路由导航守卫，及登陆判断，管理系统的权限判断
### 设计组件，组件的设计包括常用逻辑组件的设计，及菜单展示栏的基本格局设计等。
### 设计vuex存储，store的设计包括用户信息的存储，登陆信息的更新，权限信息的存储。
